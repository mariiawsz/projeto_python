# Projeto Python - Verificação de Idade

Este projeto foi desenvolvido para praticar lógica de programação em Python e versionamento com Git e GitHub.

## Funcionalidades
- Solicita o nome do usuário
- Solicita a idade
- Informa se é maior ou menor de idade
- Valida idades negativas

## Tecnologias utilizadas
- Python
- Git
- GitHub
